# GTN — Frontend

Next.js web app for end users (dialer, chat, voice rooms, settings, plans).

**Full project documentation, architecture, and setup:** see the **[repository root README](../README.md)**.

**Quick config:** set `NEXT_PUBLIC_API_URL` in `.env.local` to your API base (include `/api` if the server mounts routes there), e.g. `http://localhost:5000/api`.
